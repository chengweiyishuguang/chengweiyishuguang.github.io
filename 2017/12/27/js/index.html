1.4.3arguments

Javascript在创建函数的同时，会在函数内部创建一个arguments对象实例.

arguments是存储了函数传送过过来实参。arguments对象只有函数开始时才可用。函数的arguments 对象并不是一个数组，访问单个参数的方式与访问数组元素的方式相同

arguments对象的长度是由实参个数而不是形参个数决定的

这个单词只在函数内使用，而且是正在执行的函数。

&lt;script&gt;
// 尽量要求形参和实参相互匹配
function fn(a,b) {
// console.log(fn.length); 返回的是 函数的 形参的个数
// console.log(arguments.length); 返回的是正在执行的函数的 实参的个数

// arguments  里面存放的是 [1,2]
    if(fn. **length** == arguments. **length** )
    {
         **console**.log(a+b);
    }
    else
    {
         **console**.error(&quot;对不起，参数不匹配，参数正确的个数应该是&quot; + fn. **length** );
    }
}
fn(1,2);

&lt;/script&gt;

1.5变量提升

1.function 函数作用域里的变量会遮盖了上层作用域变量。

2.在function作用域内，变量的声明被提升了。

注意：

变量赋值并没有被提升，只是声明被提升了

var num = 10;
fun();
function fun(){
console.log(num);
var num = 20;
}
相当于转为为下面代码
var num = 10;
fun();
function fun() {
var num; // 带有var 的变量 提升到 function 作用域的最顶层
// 只提升变量 ，不给值
console.log(num);
num = 20;
}

1.6return 返回值

定义：

一个函数实际上就是一个计算过程，计算完成之后的结果就是返回值。

定义函数的返回值：

在函数内部用return来设置返回值，一个函数只能有一个返回值。

同时，终止代码的执行。

所有的自定义函数默认没有返回值

function fn() {
return 3** ; // 终止程序执行 不会执行return 下面的语句
**console.log(11);
}
fn(); // 函数返回的结果就是3
console.log(fn());

return 后面至少有一个空格，后面有什么，就返回什么。

1.9 获得焦点和失去焦点事件

得到光标和失去光标 简单理解

获得焦点： onfocus onmouseover onclick

失去焦点： onblur

这个的原理：

得到焦点：

一上来不应该直接清空。

如果用户没有输入内容就应该清空

如果用户输入了内容，就不应该清空

我们怎么知道用户有没有输入内容呢？

结论： 只要文本框里面的值不是 请输入… 就说明用户输入了

就不应该清空，否则清空

失去焦点：

如果当表单里面的值为空的话，我们就应该还原原来的值

&lt;script&gt;
var txt = document.getElementById("txt");
txt.onfocus = function() { // 获得焦点之后才去判断
// alert("您得到了焦点")
// 当我们得到了焦点的时候， 文本框里面的值为空 有点问题
// txt.value = "";
if(txt. value == "请输入…") // 说明用户还没有输入
{
txt. value = ""; // = 赋值 == 判断 === 全等
txt. style. color = "#333";
}
}
txt.onblur = function() {
// alert("您失去了焦点")
if(txt. value == "") // 如果当前的表单的值为空
{
// 回来吧
txt. value = "请输入…";
txt. style. color = "#ccc";
}
}
&lt;/script&gt;
